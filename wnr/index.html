This is the Windows Never Released archive. Essentially, it is made-up versions of Windows, usually put together in a video. There's a lot that goes into these: you need the image, the startup and shutdown sounds it uses (and occasionally some others), sometimes a version list if a video is being made, and occasionally some desktop concepts. All of these types of materials from many different creators are archived here.

Videos - There are many.
Packs - Also many.
Screens - Startup, shutdown, even some desktop and program photos.
Sounds - Startup and shutdown typically, but this collection also includes the other system sounds.
Docs - Version lists, audio guides, whatever else.
